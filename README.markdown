# Survey Form

A template Survey form is made using HTML5 and CSS3

## Getting Started

Its fairly simple to use.You should have some prior knowledge about HTML5 and CSS3 to use it.

### Prerequisites

```
A Code Editor
Basic Knowledge : HTML5 and CSS3
```

## Built With

- HTML5
- CSS3
